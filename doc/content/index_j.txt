はじめに
========

クラスについては、 `APIドキュメンテーション`_ に参照してください。

.. _`APIドキュメンテーション`:

条件
====

MobileRobotKinematicsはOpenRTM-aist 1.0.0以上のC++版が必要です。

MobileRobotKinematicsは CMake_ を使います。CMake 2.8以上が必要です。

.. _CMAke:
   http://www.cmake.org

インストール
============

インストーラ
------------

Windowsのユーザはインストーラパッケージを使用してコンポーネントをインストール
することができます。これはコンポーネント及びそのすべての必要なライブラリを
インストールします。Windowsでインストールする場合、インストーラの使用を推奨してます。

- インストーラをダウンロードしてください。
- インストールを始めるためにインストーラをダブルクリックしてください。
- 指示にしたがってコンポーネントをインストールしてください。
- 環境変数の変更を適用するため、コンポーネントを使用する前にコンピューターを
  再起動する必要があるかもしれません。

MobileRobotKinematicsは ``MobileRobotKinematicsComp`` の実行をダブルクリックして実行することが
できます。あるいは、 ``MobileRobotKinematics`` を初期化関数の ``MobileRobotKinematicsInit`` を利用して、
マネージャにロードすることができます。

ソースから
----------

ソースを使う場合は以下の手順でインストールしてください。

- ソースをダウンロードして解凍してください::

    tar -xvzf MobileRobotKinematics-1.0.0.tar.gz

- 解凍されたフォルダに入ってください::

    cd MobileRobotKinematics-1.0.0

- ``build`` フォルダを作ってください::

    mkdir build

- `` build`` フォルダに入ってください::

    cd build

- CMakeを実行してください::

    cmake ../

- エラーが出無い場合、makeを実行してください::

    make

- ``make install`` でコンポーネントをインストールしてください。選択された
  インストール場所に書き込み権限があるかを確認してください::

  ``make install``

- インストールする場所はccmakeを実行して ``CMAKE_INSTALL_PREFIX`` を
  設定することで変更が可能です。

    ccmake ../

ここまでで、コンポーネントが使えるようになりました。コンフィグレーションは次のセクションを
参照してください。

MobileRobotKinematicsは ``MobileRobotKinematicsComp`` を実行（ ``${prefix}/components/bin`` に
インストールされます）することでスタンドアローンモードで実行することができます。
あるいは、 ``libMobileRobotKinematics.so`` を初期化関数の ``MobileRobotKinematicsInit`` を利用して、
マネージャにロードすることができます。このライブラリは ``${prefix}/components/lib`` 
または ``${prefix}/components/lib64`` にインストールされます。

パラメータ       データ型           デフォルト値     意味

ポート
======

ポート名        ポート型    データ型                       意味

updatePose      InPort      RTC::TimedPose2D               
targetWheelVelocity OutPort     RTC::TimedDoubleSeq            
currentPose     OutPort     RTC::TimedPose2D               
=============== =========== ============================== ====

例
==
